# Brand Assets

Place dashboard brand files in this folder.

Primary logo candidates used by the dashboard:
- radian-forge-logo.png
- (legacy fallback) ../logo.png

Optional mark used in the robot identity card:
- radian-forge-logo-mark.svg

Expected served paths:
- /assets/brand/radian-forge-logo.png
- /assets/brand/radian-forge-logo-mark.svg

If brand files are missing, the UI falls back to a text wordmark and hides the mark without breaking layout.
